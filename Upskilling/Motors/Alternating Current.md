In all the content previously discussed, current has been flowing in one direction while electrons flow the other way. Polarity is fixed.

While this is how power sources like batteries produce current, mains power does not behave this way.

The `+` and `-` alternate rapidly at 50-60 times per second. This is known as alternating current because the current alternates. This means that electrons in a conductor move back and forth rather than steadily in a single direction.

Both voltage and current alternate direction together at the same frequency.
![[Pasted image 20260308130135.png]]

Alternating current (AC) is used mainly because it is much more efficient for transmitting electricity over long distances.
