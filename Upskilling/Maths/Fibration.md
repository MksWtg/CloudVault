
Say we have a continuous map $p : E \to B$

Where E is the total space and B is the base space

This map is a fibration if it has the homotopy lifting property

For space $X$, map $f : X \to E$, map $p : E \to B$ and homotopy $F : X \times [0, 1] \to B$ with

$p \circ f : X \to B = F(-, 0)$ 

There exists a homotopy 

$F' : X \times [0, 1] \to E$ such that

$F'(-, 0) = f$
$p \circ F' = F$

Any homotopy in B can be lifted to one in E

1) starting with any function X to E, applying F' to transform it into another function and applying p is the same as doing the function and p and then mapping with F'. We basically have a commutative diagram
