Random side note: to really be part of this community, you really need to study maths properly. Set theory, lin alg, all the way up will give the best understanding of the topic.

The only subjects that have been directly referenced are:
- propositional logic
- lambda calculus and type theory
- denotational semantics
- higher category theory
- homotopy theory
- model theory (depends on category theory)

It will be good to learn everything, e.g. abstract algebra, point set topology to better understand.

This will help us understand when e.g. in amelies blog people reference half adjoint functors.

We also need to read the HoTT book obviously.
