In OO, the smallest module of code is a class. Functionality and programs are built by multiple classes calling each other, with each class having limited responsibilities (Read: [[Encapsulation]] and Sing).


## Coupling

Coupling describes a relationship between two modules. Two modules are loosely coupled if one module can be modified without changing the other.
