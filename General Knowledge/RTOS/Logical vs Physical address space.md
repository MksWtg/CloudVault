The **logical address space** is the set of addresses generated by the CPU during program execution, which are used by processes and are not directly tied to actual memory locations. The **physical address space** is the set of real addresses in main memory (RAM) where data is actually stored after being translated from logical addresses by the memory management unit (MMU).

Logical and physical addresses are the same in compile-time and load-time address-binding schemes; logical (virtual) and physical addresses differ in execution-time address-binding scheme
