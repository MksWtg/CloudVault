Computer system can be divided into four components:
- hardware provides basic computing resources such as CPU, RAM, GPU, IO devices
- operating system controls and coordinates the hardware among various apps and users (computer has one OS that is used at a time, e.g. windows or linux)
- Application programs define the ways in which the system resources e.g. CPU are used to solve computing problems e.g. word processors allow you to write and format text like bold and underline. Browsers let you access data available on the internet.
- Users are people or machines or computers and they use the apps
