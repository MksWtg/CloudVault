## What will I learn?

This module focuses on

1. introduction to the programming of POSIX Inter-process Communication (IPC) on Unix Part 2.
2. concept description of Unix for Shared Memory, Scheduling and Memory Locking.
3. description of shm_open and mmap (memory mapping) functions, which set up the shared memory and mapping operation.
4. Introduction to POSIX.4 scheduling interfaces including parameter setting such as priority and SCHED_FF and SCHED_RR.
5. Introduction to clock function.

This forms the core component to the subject.

## What will I be able to do at the end of this module?

At the end of this module, you should be able to answer these questions under Unix and POSIX:

1. What are the shm_open, mmap and sched_param? how these functions work?
2. In scheduling interfaces, what do SCHED_FIFO, SCHED_RR and SCHED_OTHER mean?
3. Can you use your own words to describe the FCFS scheduling, RR scheduling?
4. Brief to describe the clock function and locking block memory for related process.
