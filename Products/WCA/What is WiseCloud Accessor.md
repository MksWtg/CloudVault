## The Problem

All CWC solutions exist to solve a problem. The problem is for hosted customers, we run cargowise on our servers but how does the user use them if they run on our servers?

## The Solution

WCA is both a client side interface for users to initiate a session and other server side softwares that connect the user to a running cargowise instance.
